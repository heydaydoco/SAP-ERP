# @erp/trade-data

Reference data for trade-compliance. Loaded by `data-migration` and the
`hs-classification` / `fta-origin` / `compliance-screening` modules.

```
hs/    HS code master CSVs (HSK 10-digit), duty rates
fta/   FTA agreement tables + PSR (Product Specific Rules)
spl/   SPL / strategic-goods / sanctions screening lists
```

Empty during scaffolding — populated in Phase 7 (trade-compliance).
